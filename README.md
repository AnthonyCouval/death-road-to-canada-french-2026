# death-road-to-canada-french-2026
Traduction française du jeu Death Road To Canada avec les derniers fichiers de 2026 à jour

Comment installer ? 

Il suffit de remplacer le dossier deathforth de votre repertoire de jeu par celui de ce repo, il faut tout écraser.
Votre dossier devrait se trouver dans **C:\Program Files (x86)\Steam\steamapps\common\DeathRoadToCanada**
